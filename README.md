# Recap Car Rental Company - Rent  A Car

This project is an online car rental project for  [kodlama.io](https://www.kodlama.io/) online course program of Engin Demirog and his team.

### Tecnologies
- Angular 11
- TypeScript
- Bootstrap 5
- Jquery

### FrontendTechniques
- Pipe
- JWT
- Service
